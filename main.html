Головна

Дякую що завітали на мій сай візитку. Даний сайт для вираження рівня моїх знань HTML, CSS та JS. Як видно зняння не глубокі, знаю тільки основи, ніколи не працював веб розробником, темою веб програмування почав цікавитися коли почув про набір на курси JavaSolution. Надіюся з часом сайт ставатиме кращим та більi функціональний по мірі мого подальшого освоєння HTML, CSS та JS.

Сайт не виконано з використанням Meteor так як поки що займаюся освоєнням основ HTML, CSS та JS, але на даному етапі вже вдалося здобути базові теоретичні та практичні навички, тому найближчим часом постараюся продемонструвати якісь напрацювання в Meteor.
